1. Instance segmentation in the wild with active learning
The idea here is to build an instance segmentation model (it draws a separate mask for each object instance, 
e.g., each bottle, each chair), but instead of labeling thousands of masks up front, we do active learning.
That is for example we start with a tiny labeled set (e.g., 50–150 images) and train a baseline segmentation model.
Then we run it on a big pool of unlabeled images and use a “smart picker” to choose the next batch of images that would help most if labeled.
Then we label only those and retrain / fine-tune and we repeat this process for several rounds.
At the end we try to highlifht a curve between accuracy and number of labeled masks/images.

2. Video enhancement / deblurring for phone videos
Phone videos often look bad because of low resolution, digital zoom, hand shaking, low light, compression (whatsapp, instagram, etc), etc.
The main idea is to make a video enhancer that improves quality. Two main directions:

- Direction A — Video Super-Resolution (VSR)
Input: low-res video (e.g., 480p)
Output: higher-res (e.g., 720p or 1080p)
Core challenge: create new detail without inventing nonsense; avoid flicker.

- Direction B — Video Deblurring / Denoising
Input: blurry/noisy video
Output: sharper/cleaner video at same resolution
Core challenge: sharpen without ringing/ghosting; keep temporal stability.
